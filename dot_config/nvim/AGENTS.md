# Neovim Configuration Instructions

## Neovim Documentation

- When changing files under `dot_config/nvim/`, review
  `dot_config/nvim/README.md` and update it in the same change when user-facing
  configuration, key mappings, plugins, or common operations change.
- Keep `dot_config/nvim/README.md` concise and written in English.
- Document custom behavior and representative plugin operations. Do not
  document standard Vim commands that are not customized by this repository.
- Keep `AGENTS.md`, `CLAUDE.md`, and `README.md` in this directory source-only
  by preserving their entries in `.chezmoiignore`.
