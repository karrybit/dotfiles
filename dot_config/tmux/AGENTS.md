# tmux Configuration Instructions

## Documentation

- When changing files under `dot_config/tmux/`, review
  `dot_config/tmux/README.md` and update it in the same change when user-facing
  configuration, key bindings, plugins, or common operations change.
- Keep `dot_config/tmux/README.md` concise and written in English.
- Document custom behavior and representative plugin operations.
- Keep `AGENTS.md`, `CLAUDE.md`, and `README.md` in this directory source-only
  by preserving their entries in `.chezmoiignore`.
